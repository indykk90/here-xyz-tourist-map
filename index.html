Introduction

This training was prepared to show functionalities of the HERE XYZ Studio. HERE XYZ Studio is an interactive, visual, web-based application for accessing XYZ Hub data and creating maps within minutes. This tutorial doesn't require coding skills.

Training objectives

In this tutorial, you're going to build a tourist map for Central European countries: Baltics, Poland, Czech Republic, Slovakia and Hungary. This map will:

  • Display borders of each country
  • Display collections from wego.here.com
  • Allow you to draw and display new places on a map
  • Allow you to style your layers any way you want it
  • Create and display bookmarks

Useful materials:

  • https://www.here.xyz/studio/
  • https://www.here.xyz/cli/
  • https://wego.here.com/

What you'll learn

  • How to create an account in XYZ Studio
  • How to install HERE Data Hub CLI
  • How to add larger spatial data to a project using Data Hub CLI
  • How to add collections from wego.here.com to the HERE XYZ Studio
  • How to display data on a map
  • How to publish your map

Install node.js

Note: In this tutorial we are going to upload Shapefiles, which are larger files than GeoJSON or CSV. To do so, we need to use Data Hub CLI.

Node.js is required to install HERE CLI. To use it, you should have npm installed. The best way is to go to nodejs.org and install the appropriate package for your system. Download the package and install it normally.

Install HERE CLI

Install the latest version of HERE CLI with following command:

npm install -g @here/cli

Type it in Node.js command prompt.

Configure HERE CLI

HERE CLI needs to have access to your projects to help you work with your data. For that, you need to log in using your HERE developer account. You only need to do this once for a project.

here configure account

Type it in Node.js command prompt.

When prompted, enter your email and password that you use in the HERE Developer portal.

Note: For more details visit https://www.here.xyz/cli/

Create new project

When you first sign in you won't see any projects, so to create a new one, click Create new project button or click on the circle icon with + inside (which is located below Create new project button).

In the next step you can give your map a name and also a description.

In this step you can also pick a base map. Feel free to pick the one you like the most, there are a few available at the moment. We will go with the one from the base maps provided by HERE. Please, keep Show labels ticked.

Download required shapefiles

There are two ways of adding spatial data to the project:

  • Directly by HERE XYZ Studio
  • Using Data Hub CLI

We are going to play with Shapefiles, so for this reason, Data Hub CLI is required.

Adding data by using Data Hub CLI

Note: For more information about Data Hub CLI, visit: https://developer.here.com/tutorials/using-the-xyz-cli/

To add larger data to the HERE XYZ Studio, we need to use Data Hub CLI. First of all, open Node.js Command Prompt.

Note: This step was mentioned previously, if you already provided your HERE credentials, skip this step and jump to the next one - uploading shapefiles.

In the beginning, please provide HERE credentials. Enter your email credentials by typing:

here configure account

You will be prompted to enter your email address and password. Your email will be echoed, but there will be no feedback when you paste in your password.

Next step is to upload our first Shapefile (CEU_countries.shp), type:

here xyz upload -f CEU_countries.shp

You will be asked to provide title and description for the new space.

After this, your shapefile will be uploaded.

To upload another Shapefile (Places_to_visit.shp), follow the same rule.

In this step, we are going to export collections from the Here WeGo. Create an account or use an existing one.

Uploading CSV to HERE XYZ Studio using Data Hub CLI

To upload csv file to the Data Hub CLI, type in command prompt:

here xyz upload -f my_collections.csv

You will be asked to provide property for Latitude and Longitude in your csv file. Just select relevant columns. After this, your csv file will be added to your XYZ Space.

Now you can go to the Data section in XYZ Studio to check properly uploaded data.

Updating the map

Now it's time to select and display places we would love to visit in the future. It's the last missing element of our map.

In this step, we will update our map of the new layer - places we want to visit in the future. This layer has been already added to our map (Places to visit). Now it's time to add some places on a map.

To do this, click on the Places to visit layer and toggle edit mode (icon of pen), which is located in the top left corner of your screen.

To add a new point, click a plus button and select point.

Click on a map (on a place you want to visit) and save your edit.

When your place is drawn, go to your Data Space and click on the layer we are editing (Places I want to visit). Open attribute table and update columns. Provide information about the type of place and year when you would like to visit this spot.

Depending on the attributes, we can create a new style group which takes into account date and type of place.

Bookmarks

If you want to switch between different predefined map views, you can define your own location bookmarks and quickly switch between them. Let's use an example of our map. We can add two bookmarks to switch between countries, one for Poland and another one for Slovakia.

To do this, zoom to the view you want to save and click the Bookmark current view button located in the top right corner and provide a name for it.

To use a new bookmark, search for it in a search bar. It will zoom you to the saved bookmark view.

Now it's your turn, add bookmark for Poland.

Now it's time to publish our map. You can easily share a link to your map or copy & paste a snippet of code that you can directly include on a web page or in a blog for example.

By default, every map you create with HERE XYZ Studio is private and visible only to you until you decide to publish it.

All you have to do to publish the map is to turn publishing on (you can turn it off anytime again as well if you wish). This button is located in the bottom left corner of your screen.

When the map is published, you also see the URL where it can be reached. You can copy this and share it with people directly or you can link to it from somewhere like a blog or wiki page.

You can also copy the EMBED code and use it directly in the code of your website or blog. It's a bit of HTML that will show your map directly on the page.

Share your map with the world! :-)
